# Moikka👋

Olen toisen vuoden ICT-insinööriopiskelija Metropoliassa. Keskityn opinnoissani ja projekteissani laaja-alaiseen Full-Stack -web-kehitykseen, tietokanta-arkkitehtuureihin sekä tekoälyä hyödyntäviin backend-ratkaisuihin. 

Koodausfilosofiani nojaa vahvasti **80/20-sääntöön**: teen 80 % työstä – arkkitehtuurin, ydinlogiikan ja ongelmanratkaisun – itse manuaalisesti pitääkseni langat tiukasti omissa käsissäni. Tekoälyä hyödynnän työkaluina sen 20 % rutiinityön ja mekaanisen koodin generoinnin nopeuttamiseen. En ulkoista ymmärrystäni koneelle.

## 🚀 Pääprojektit

### 1. ByrokratianKuormituksenV-hennys (Bureaucracy Decoder AI)
**Teknologiat:** `FastAPI`, `Python`, `React`, `TypeScript`, `NLLB (AI)`, `OCR`
* Lokaali, yksityisyyttä kunnioittava full-stack-sovellus, joka on suunniteltu lukemaan ja kääntämään monimutkaisia suomalaisia viranomaisasiakirjoja ymmärrettävälle kielelle.
* Rakensin sovellukseen älykkään tekstin palastelun (Context Merging), joka estää tekoälyn hallusinaatiot, ja integroin lokaalin kielimallin takaamaan 100 % tietosuojan ilman pilvipalveluita.

### 2. Quantix Logistics
**Teknologiat:** `React`, `Node.js`, `Express`, `MySQL`, `TypeScript`
* Logistiikka-alan tarpeisiin suunniteltu web-sovellus, joka toteutettiin tiimityönä.
* Toimin osana kehitystiimiä vastaten full-stack-toteutuksesta: API-integraatioista, tietokantaratkaisuista (MySQL) sekä käyttöliittymän rakentamisesta.

### 3. CCTV-AI-Security-Monitor
**Teknologiat:** `Python`, `AI / Computer Vision`, `Linux`
* Tekoälyä hyödyntävä turvallisuusmonitorointityökalu. Sovellus on suunniteltu valvontakameradatan älykkääseen analysointiin ja poikkeamien havaitsemiseen Python-pohjaisilla työkaluilla.

### 4. Defense-Detection
**Teknologiat:** `TypeScript`, `JavaScript`, `Node.js`
* Tietoturvaan ja järjestelmien puolustukseen keskittyvä tunnistustyökalu. Rakennettu modernilla TypeScriptillä ja JavaScriptillä tehokkaan ja tyyppiturvallisen logiikan varmistamiseksi.
