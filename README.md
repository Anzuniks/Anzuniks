# Moikka 👋

Olen toisen vuoden ICT-insinööriopiskelija Metropoliassa. Keskityn opinnoissani ja projekteissani laaja-alaiseen Full-Stack -web-kehitykseen, tietokanta-arkkitehtuureihin sekä tekoäly- ja koneoppimisratkaisuihin. 

Koodausfilosofiani nojaa vahvasti **80/20-sääntöön**: teen 80 % työstä – arkkitehtuurin, ydinlogiikan ja ongelmanratkaisun – itse manuaalisesti pitääkseni langat tiukasti omissa käsissäni. Tekoälyä hyödynnän työkaluina sen 20 % rutiinityön ja mekaanisen koodin generoinnin nopeuttamiseen. En ulkoista ymmärrystäni koneelle.

## 🚀 Pääprojektit

### 1. ByrokratianKuormituksenV-hennys (Bureaucracy Decoder AI)
**Teknologiat:** `FastAPI`, `Python`, `React`, `TypeScript`, `NLLB (AI)`, `OCR`
* Lokaali full-stack-sovellus viranomaisasiakirjojen kääntämiseen. Integroin lokaalin kielimallin ja kehitin algoritmin älykkääseen tekstin palasteluun (Context Merging), joka takaa tarkan ja tietoturvallisen lopputuloksen.

### 2. Kauppakeskussimulaattori
**Teknologiat:** `Java`
* Oliopohjainen simulaatioprojekti, jossa keskityin tietorakenteiden hallintaan, oliosuunnitteluun ja simulaatiologiikan optimointiin.

### 3. Quantix Logistics
**Teknologiat:** `React`, `Node.js`, `Express`, `MySQL`, `TypeScript`
* Logistiikka-alan web-sovellus. Vastasin koko stackin integraatiosta: API-rajapinnoista, tietokantasuunnittelusta ja frontendin toteutuksesta.

### 4. CCTV-AI-Security-Monitor
**Teknologiat:** `Python`, `Scikit-learn`, `Computer Vision`
* Tekoälypohjainen valvontatyökalu poikkeamien havaitsemiseen. Käytin Pythonia ja Scikit-learnia datan käsittelyyn ja mallien kehittämiseen valvontakameradatan analysointia varten.

### 5. Defense-Detection
**Teknologiat:** `TypeScript`, `JavaScript`
* Tietoturvaan ja järjestelmien puolustukseen keskittyvä tunnistustyökalu, joka hyödyntää tyyppiturvallista logiikkaa järjestelmien tunnistamiseen ja suojaamiseen.
